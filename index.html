Hi, my name is David

and I have been a Web Developer for 6 years. I love code and all the challenges and possibilities it offers.

Passion For Coding

Coding is powerful.

It's the only tool you need to solve challenging problems, create helpful utilities and meaningful experiences. Code isn't just the hammer, but also the tools you make the hammer with. To me,it is unparalleled in its versatility and utility.

"Any fool can write code that a computer can understand. Good programmers write code that humans can understand."

Martin Fowler

More than being passionate about what you can do with programming, I'm passionate about the code itself. It is an expression of creativity, not unlike a form of art, and it's beauty lay in its structure; how communicative, clear, and efficient it is are aspects of its aesthetics.

Passion For Learning

The investment you make in yourself.

Few things are as rewarding as learning a new technology or technique that you can leverage to actual benefit in your daily life; be it personal or professional. Whether it is SVG Filters, Service Worker, or graph traversal algorithms, I find learning to be a fun activity in its own right and that it makes me a more well rounded developer. I try to spend at least a few days a month learning something new and find I thrive in an atmoshpere which encourages this personal growth.

Passion For User Experience

Putting the user first.

One of the things which I find the most reward in is creating an interesting and fun user experience. Spending excessive amounts of time tweaking the easing of an animation until it feels "just right." But user experience is more than just flashy animations and effects on high-end desktops. It's the performance of animations, keeping them as close to 60FPS and jank free. Keeping Page load speed fast and responsive. Making the content accessible to a wider audience, including those with disabilities or impairment.
